Detailed Description

Represents a pool of SIPP nodes. This class manages the allocation and deallocation of SIPP nodes. Saves a lot of time. If there are not enough nodes in the pool, it allocates more and stores them in a list, which is then merged into the pool at the end of the iteration. Once a node is allocated, it is not deallocated until the pool is destroyed.
