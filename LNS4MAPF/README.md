# LNS4MAPF
Code for my diploma thesis about using Large Neighbourhood Search for Multi-Agent Pathfinding Problem.

# Contents
The repository contains the following files:
- README.md - this file with information about the project
- MAPF-benchmark/ - a dataset of MAPF instances
- my_map/ - a dummy map and scene I created for testing purposes
- my_code/ - implementation of my MAPF solver based on Large Neighborhood Search combined with Prioritzed Planning and Safe Interval Path Planning algorithm
- experiments/ - the directory with scripts for visualization of the experiment results
